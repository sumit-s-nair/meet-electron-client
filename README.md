# Meet Electron Client

Desktop meeting client built with React, Vite, and Electron.

## What This App Does

- Provides a desktop shell around a meeting UI.
- Uses your backend API for guest users and meeting/token creation.
- Uses LiveKit for real-time audio/video.
- In packaged mode, Electron loads the hosted app URL:
  - `https://meet-lite.vercel.app/`

## Prerequisites

- Node.js 20+ (recommended: latest LTS)
- pnpm 10+
- Windows/macOS/Linux (Electron)

## 1 Install Dependencies

```bash
pnpm install
```

## 2 Configure Environment Variables

Create a `.env` file at project root (or update the existing one):

```env
VITE_API_URL=https://your-api.example.com
VITE_LIVEKIT_URL=wss://your-livekit-instance.livekit.cloud

# Optional for local Electron dev override
VITE_DEV_SERVER_URL=http://localhost:5173

# Optional for packaged Electron override
ELECTRON_START_URL=https://meet-lite.vercel.app/
```

Notes:

- `VITE_API_URL` is used by the React app for guest user and meeting/token endpoints.
- `VITE_LIVEKIT_URL` is passed to LiveKit for RTC connection.
- If `ELECTRON_START_URL` is not set, packaged builds default to `https://meet-lite.vercel.app/`.

## Backend Setup

Set up and run the backend from:

- https://github.com/sumit-s-nair/meet-realtime-backend

Use that backend's base URL as `VITE_API_URL` in this client app.

LiveKit API key and secret setup are also explained in that backend repository. Follow those instructions to configure backend environment variables before testing meeting creation/join flows.

## 3 Run the App (Development)

### Web only (Vite dev server)

```bash
pnpm run dev
```

### Electron + Vite together

```bash
pnpm run start
```

This starts Vite on `http://localhost:5173` and then launches Electron.

## 4 Build

### Build frontend bundle

```bash
pnpm run build
```

### Build Electron main process

```bash
pnpm run build:electron
```

### Create packaged desktop app

```bash
pnpm run electron:build
```

Packaged app output is generated by `electron-builder` (typically under a release/output folder based on platform settings).

## 5 Use the Packaged App

After installing/running the packaged app:

- Electron opens the hosted URL by default: `https://meet-lite.vercel.app/`.
- Camera/microphone/screen permissions are requested by Electron.
- On Windows, system audio loopback is enabled for screen capture when requested.

## Available Scripts

- `pnpm run dev`: Start Vite dev server.
- `pnpm run electron`: Launch Electron directly using `electron/main.ts`.
- `pnpm run start`: Run Vite and Electron together for desktop development.
- `pnpm run build`: Build React/Vite app.
- `pnpm run build:electron`: Compile Electron main process.
- `pnpm run electron:build`: Build everything and package app with `electron-builder`.
- `pnpm run lint`: Run ESLint.
- `pnpm run preview`: Preview production web build locally.

## Troubleshooting

- If `pnpm run dev` fails, run `pnpm install` again and check your Node.js version.
- If meeting join/token calls fail, verify `VITE_API_URL` and that backend endpoints are reachable.
- If media devices are not available, check OS-level camera/mic/screen permissions for Electron.
- If packaged app should open a different site, set `ELECTRON_START_URL` before packaging.
